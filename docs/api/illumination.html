illumination Module¶

class volume_grating.illumination.Playback(source)¶

Bases: object

Playback class creates an illumination instance that emulates how a hologram will played back. Therefore, it needs only one light source.

Parameters

source – an instance of Source subclasses.

class volume_grating.illumination.Record(source1, source2)¶

Bases: object

Record class creates an illumination instance that emulates how a hologram will be made. Therefore, it needs two light sources as its argument.

Parameters

  • source1 – an instance of Source subclasses.

  • source2 – an instance of Source subclasses.

class volume_grating.illumination.TargetPlayback(source1, source2)¶

Bases: volume_grating.illumination.Record

TargetPlayback class creates an illumination instance that emulates how a hologram should playback in an attempt to design a recording process. Therefore, it needs two light sources. Due to the same requirement as the Record class, it is convenient to make TargetPlayback a subclass of Record.

Parameters

  • source1 – an instance of Source subclasses.

  • source2 – an instance of Source subclasses.
