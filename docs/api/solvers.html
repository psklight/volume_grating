run(xs_init, points, target, points_weight=array([1.]), method=None)¶

Runs the candidate search by minimizing the loss/error value.

Parameters

  • xs_init – (N,) ndarray values specifying the initial guess of the parameters to optimize

  • points – a list of points on the hologram to optimize, must be a list of sympy.vector.Point.

  • target – a list of target response, such as grating vectors or diffraction efficiency, with the same length as that of points argument.

  • points_weight – an (N,) ndarry of weight for each hologram points. If a scalar number if provided, the same weight is applied to all points. Default is one.

  • method – [optional, Default=None] a method that scipy.optimize.minimize uses. If None, self.optim_method will be used.

Returns

a return from scipy.optimize.minimize()

class solvers.Response(hologram, playback=None, engine=None, order=1)¶

Bases: object

Response is a base instanceable class to compute responses of a hologram. playback is needed when any diffraction responses are queried. engine is needed when diffraction efficiency is queried.

Parameters

  • hologram – an instance of HologramBase subclasses only.

  • playback – [default = None] an instance of illumination.Playback.

  • engine – [default = None] an instance of Engine subclasses that defines physics used to calculate diffraction efficiency.

  • order – an integer to specify a diffraction order to compute.

get_k_diff_at_points(points, order=None, materail_outside=None)¶

Return a list of diffracted k vector against a list of Point instances for a specified order. If the order is left None, the order value would be that of the solver instance. The environment outside a hologram is defined by a Material instance in material_outside. If material_outside is None, the returned diffracted k vector is inside the hologram. Otherwise, Snell’s law will be applied to return diffracted k vectors that are already refracted into the environment.

Parameters

  • points – a list of sympy.vector.Point instances.

  • order – [optional, default=None] a diffraction order, must be an integer. If None, orders will be set to order of the solver instance.

  • material_outside – [default=None] an instance of Material subclasses, representing an environmental material.

Return [k_diff]

a list of sympy.vector.Vector instances.
