class sources.Planewave(material, direction, wavelength=6.6e-07)¶

Bases: sources.Source

Planewave class defines a planewave source, whose wave vector is the same every where.

Parameters:
  • material – an instance of Material or its subclasses, specifying which material the source is embedded.
  • direction – an (3,) ndarray specifying x, y, z components of direction vector. It does not need to be normalized.
  • wavelength – a numerical value of wavelength in meter. Default = 660 nm.

class sources.PointSource(point, material, direction='diverge', wavelength=6.6e-07)¶

Bases: sources.Source

PointSource class defines a source that light either diverges or focuses to a point.

Parameters:
  • point – a (3,) ndarray or a list of 3 numeric values.
  • material – an instance of Material or its subclasses, specifying which material the source is embedded.
  • direction – a string, either “diverge or “focus”.
  • wavelength – a numerical value of wavelength in meter. Default = 660 nm.

class sources.Source(material, wavelength=6.6e-07)¶

Source is a base class for all optical source classes.

Parameters:
