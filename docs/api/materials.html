materials Module¶

class volume_grating.materials.Cauchy_2coeff(coeff)¶

Bases: volume_grating.materials.Material

Cauchy_2coeff models a material with two Cauchy coefficients, B and C. See https://en.wikipedia.org/wiki/Cauchy%27s_equation for Cauchy model. Unit of C is micron**2.

Parameters

coeff – (2,) ndarry of numbers. B = coeff[0] and C = coeff[1].

n(wavelength)¶

Returns a (real-part) of refractive indices at specified wavelengths.

Parameters

wavelength – a list or ndarray of wavelengths in meter.

Returns

refractive index n

class volume_grating.materials.NKconstant(N, K=0)¶

Bases: volume_grating.materials.Material

NKconstant models a material with a constant complex index of N + i*K.

Parameters

  • N – the refractive index value

  • K – the absorption/gain value

Return the whole complex refractive index, i.e. n+i*k, which is constant
