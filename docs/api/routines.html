volume_grating.routines.fit_de_spectral(wavelengths, de_data, solver, update_solver, initial_guess, bounds, plot=True)¶

This function fits a data of diffraction efficiency with prediction calculated via solvers.Response object.

The update_solver_func(params, solver) must be a callable with two arguments, in order, params and solver. These arguments are the same as for objective function.

For example, the following code will fit data with prediction by varying dn and thickness of a hologram. .. code-block:: python

# wavelengths are defined in nm # de are 1D numpy array from experimental result.

def update_solver(params, solver):

dn = params[0] t = params[1] solver.hologram.dn = dn solver.hologram.thickness = t

x0 = np.array([0.035, 1.5e-6, 0.0]) # x0 has 3 elements because the last element is used to DC-offset.

result = fit_de_spectral(wavelengths*1e-9, de, solver, update_solver, x0, bounds=((0.01, 0.08), (1e-6, 2e-6), (-0.2, 0.2)))

Parameters

  • wavelengths – numpy array for wavelengths in meter.

  • de_data – numpy array for ground-truth or measured diffraction efficiency

  • solver – an instance of solvers.Response

  • update_solver – a callable function with two arguments params and solver, see explanation above.

  • initial_guess – the initial guess for params in update_solver.

  • bounds – a tuple where each item is of the form (min, max), indicating min and max values for each parameter in params

  • plot – True or False to plot the fitting result.

Returns

fitting result (from scipy.optimize.minimize)

volume_grating.routines.get_de(solver, squeeze=True, **kwargs)¶

This function returns diffraction efficients of a given solver object and **kwargs for it. It is a convenient functional representation for other purposes such as optimization.

Parameters

  • solver – an instance of solvers.Responses.

  • squeeze – True or False whether to remove singleton dimension of the diffraction efficiency result.

  • kwargs – **kwargs that works for solvers.Response.get_efficiency_at_points().

Returns

diffraction efficiencies in numpy array.

volume_grating.routines.objective(params, truth, solver, update_solver_func, get_de_func)¶

This is an objective function (or a loss function) for fit_de_spectral to minimize mean-squared-error (MSE) loss.

params is used to update solver via updated_solver_function. Then, the predicted diffraction efficiency is calculated, via get_de_func. The MSE is calculated between the ground truth DE and the predicted DE.

The update_solver_func(params, solver) must be a callable with two arguments, in order, params and solver. These arguments are the same as for objective function.

get_de_func(solver) should only take one argument. It should return a numpy array for diffraction efficiencies.

Parameters

  • params – a 1D numpy array. The last element of ``params`` is reserved to a DC offset.

  • truth – a numpy array with the same dimension as the output from get_de_func.

  • solver – an instance of solvers.Response class.

  • update_solver_func – a callable function with two arguments params and solver, see explanation above.

  • get_de_func – a callable function that has only one argument, being solver.

Returns

MSE value

volume_grating.routines.sweep(xs, points, update_func, solver, solver_funcs, verbose=False)¶

This functions generalizes parameter sweeping to extract responses from a hologram.

Given an ndarray xs of shape (M, N). M is the number of parameter sets to sweep. N is the number of parameters in each set.

The update function update_func takes xs[i] and a solver and it updates the system according to the new parameter set xs[i], and returns an updated solver. That is, the update_func should have a structure like

def update_func(xs, solver):
    # xs is a set of parameters needed to update a solver
    # update solver using xs

    return solver

For each parameter set xs[i], the sweep function update the solver using update_func , runs each solver_func for all points, and collects the result in outputs.

Parameters

  • xs – (M, N) ndarray of parameters to sweep. M is the number of parameter sets to sweep. Each set has N parameters.

  • points – a list of sympy.vector.Point instances representing points of interest on a hologram. Np=len(points).

  • update_func – a function that updates a solver according to a single parameter sweep.

  • solver – an instance of solvers.Response.

  • solver_funcs – a list of functions/methods from a solvers.Response class (not instance). Nf=len(solver_funcs).

  • verbose – [Default = False] True or False to print out progress.

Return outputs

(Np, Nf, M) ndarray
